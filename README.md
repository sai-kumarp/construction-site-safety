# construction-site-safety
Construction Site Safety is a beginner-friendly Python project that simulates basic safety checks at a construction site by verifying the use of protective equipment such as helmets and safety vests. The project demonstrates fundamental programming concepts and promotes awareness of workplace safety practices.
